RON GitHub Download Guide

Step 1

Download and Install Github Desktop

Step 2

Clone the RoN Repository to this path on Windows: "C:\Users\%USERNAME%\Documents\Paradox Interactive\Hearts of Iron IV\mod".

Debugging Step 2

If you're unable to clone the repository download this file and put it into: "C:\Users\%USERNAME%\Documents\Paradox Interactive\Hearts of Iron IV\mod\Rise-of-Nations".

Step 3

Download the mod descriptor file and put it into the: "C:\Users\%USERNAME%\Documents\Paradox Interactive\Hearts of Iron IV\mod" folder and replace "%USERNAME%" in the file with your local account name on your device.

Results

You should have these two files in your mod folder

Play

Open your HOI4 Paradox Launcher and select Rise of Nations with the folder icon (not steam logo or RoN logo icon)
